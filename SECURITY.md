# Security Policy

## Scope
Sentinel-OOB focuses on alert delivery integrity and endpoint liveness.
It does not provide malware detection or prevention.

## Reporting Issues
Security issues or design flaws can be reported via GitHub Issues.
Do not include sensitive data in reports.

## Limitations
Sentinel-OOB does not guarantee protection against:
- Kernel-level compromise
- Persistent admin-level attackers
- Firmware or boot-level attacks
